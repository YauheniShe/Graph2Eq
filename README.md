# 🐳 Запуск генерации датасета в Docker

Этот Docker-образ предназначен для автономной генерации тренировочных данных (символьной регрессии) на сервере. 

## 1. Сборка образа
Находясь в корне проекта (где лежит `Dockerfile`), выполните команду:
```bash
docker build -t graph2eq-gen .
```

## 2. Запуск генерации
⚠️ **Важно:** Обязательно используйте флаг `-v` для проброса папки. Иначе все сгенерированные чанки (`.pkl.gz`) навсегда удалятся при остановке контейнера!

```bash
docker run -d \
  --name data_generator \
  -v $(pwd)/data:/app/data \
  graph2eq-gen
```
* `-d` — запускает процесс в фоне (вы можете безопасно закрыть SSH/терминал).
* `-v $(pwd)/data:/app/data` — физически сохраняет данные в папку `data` на вашем сервере.

## 3. Мониторинг и управление

Посмотреть логи в реальном времени (увидеть прогресс-бар `tqdm` и скорость):
```bash
docker logs -f data_generator
```
*(Для выхода нажмите `Ctrl+C` — генерация при этом **не** остановится).*

Посмотреть потребление CPU и оперативной памяти:
```bash
docker stats data_generator
```

Остановить генерацию:
```bash
docker stop data_generator
```

Запустить остановленную генерацию заново:
```bash
docker start data_generator
```

Удалить контейнер (если хотите пересобрать образ):
```bash
docker rm -f data_generator
```
